frontend

What the user sees.

It's the HTML , CSS, and JavaScript that runs on the user's browser.

javascript

The programming language of the web.

It allows you to manipulate the already-running page hence providing rich user experiences.

framework

A third party library.

It exposes utility functionalities and a clear guidance on how to build your JavaScript application.

interactive

Rich user experiences.

Your application is able to react to user input and update the screen dynamically - The look and feel of mobile devices.

THE DIFFERENT WAYS OF USING VUE

You can use Vue to control parts of HTML pages to make those parts more reactive and interactive.

Vue can also be used to control the entire page. The so-called single page application approach.
